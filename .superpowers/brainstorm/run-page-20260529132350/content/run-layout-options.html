Run 页面改造方向

目标：一进入 Run 就看到地图，并且在 Run 页直接选地图、执行地图。

Run
Map : V03 Mode: Solve
State: Idle Last: 12ms
K1/K2 Map K3 Execute
K2L Mode K4 Home

A. 直接工作台 推荐

Run 页就是地图预览页。K1/K2 直接切地图，K3 立即按当前模式执行。最快，最符合“打开就能看到地图”。

Run
> Map V03
Mode Solve
Execute
K1/K2 Cursor K3 Enter
Map item: K1/K2 change?

B. 预览 + 菜单混合

保留 Map/Mode/Execute 三项，同时在下方常驻地图。结构清楚，但按键语义会变复杂。

Run V03
Mode: Solve Save: Dirty
Execute: K3
K1/K2 Map K3 Execute
K4 Home K4L Home

C. 地图优先大预览

把地图放到最显眼位置，状态压到底部。适合主要看地图，但文字状态会更少。
